# 📋 我的任务面板（Vanilla Task Dashboard）

> 一个**零框架、纯原生**的前端实战项目，仅用 HTML/CSS/JavaScript 实现完整功能。

[👉 在线预览](https://你的用户名.github.io/vanilla-task-dashboard/)

## ✨ 功能亮点
- ✅ **任务管理**：添加/删除待办事项
- ✅ **数据持久化**：`localStorage` 刷新不丢失
- ✅ **实时天气**：调用 `wttr.in` API（无需 Key）
- ✅ **响应式设计**：桌面/手机自适应
- ✅ **交互动画**：任务增删流畅反馈
- ✅ **无障碍支持**：ARIA 标签 + 语义化结构
- ✅ **高性能**：Lighthouse 四项评分 ≥90

## 🛠 技术栈
- **HTML5**：语义化标签、表单验证
- **CSS3**：Grid / Flexbox / 动画 / 媒体查询
- **JavaScript**：DOM 操作、事件委托、模块化、Fetch API
- **工具**：Chrome DevTools、Lighthouse
